# OutLaw VPN - Документация проекта

## 📋 Описание проекта

Полноценный VPN-сервис с веб-интерфейсом, системой оплаты и автоматическим управлением подписками на базе 3x-UI панели.

## 🚀 Функциональность

### Для пользователей:
- **Регистрация и авторизация** с подтверждением по email
- **Личный кабинет** с балансом и историей подписок
- **Пополнение баланса** через Crypto Bot (USDT)
- **Покупка VPN-подписок** разных тарифов
- **Управление подписками**: включение/выключение, замена ключа, удаление
- **Автоматическое продление** подписок
- **Интеграция с V2Ray** (VLESS Reality протокол)

### Для администраторов:
- Управление через 3x-UI панель
- Промокоды для бонусов
- Мониторинг подписок

- Пароли хэшируются через SHA256
- Сессии для авторизации пользователей
- SSL для API запросов
- Автоматическое отключение просроченных подписок

## 📱 Интерфейс пользователя

### Основные разделы:
1. **Кабинет** - профиль, баланс, подписки
2. **Купить** - выбор тарифа
3. **О нас** - информация о сервисе
4. **Поддержка** - контакты и помощь
5. **Настройки** - инструкции по подключению

### Возможности управления подпиской:
- Просмотр оставшихся дней
- Копирование ссылки для подключения
- Включение/выключение VPN
- Замена ключа (при проблемах)
- Удаление подписки
- Продление истекшей подписки

## 💰 Система оплаты

1. Пользователь выбирает сумму пополнения
2. Создается счет в Crypto Bot (USDT)
3. Пользователь переходит по ссылке в Telegram
4. После оплаты баланс автоматически пополняется
5. Курс: ~1 USDT = 78 RUB

## 📝 Инструкция для пользователей

1. Зарегистрироваться (подтвердить email)
2. Пополнить баланс через Crypto Bot
3. Выбрать тариф и купить
4. Скопировать ссылку из профиля
5. Вставить в V2Ray клиент (например, V2RayTun)
6. Подключиться!

## 🔄 Автоматические процессы

- Проверка истекших подписок (каждый час)
- Очистка старых платежей (каждые 5 минут)
- Обновление статуса подписок в реальном времени
